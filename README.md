# javaweb开发者论坛系统

#### 介绍
javaweb开发的论坛系统，采用bootstrap框架绘制页面，本项目源码提供学习使用，如有不足自行更改。

#### 软件架构
软件架构说明


#### 安装教程

1.  Tomact
2.  mysql

#### 使用说明

1.  直接导入然后运行tomact就行了
2.  开源的是1.4.0版本非1.5.0（就是优化了pv，新添加手机短信验证，开发了聊天室功能，页面自适应什么的。为啥不开源勒因为找不到了）
